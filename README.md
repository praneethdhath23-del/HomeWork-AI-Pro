# 📚 Homework AI Pro

A modern, futuristic AI-powered Homework Assistant built with **HTML, CSS, and JavaScript**, powered by the **Google Gemini API**.

It helps students solve homework, explain concepts, answer questions from images, generate quizzes, and much more—all from a beautiful animated interface.

---

## ✨ Features

- 🤖 Google Gemini AI Integration
- 📖 Step-by-Step Homework Solutions
- 📝 Detailed & Short Answer Modes
- 📷 Image Homework Solver
- 🎤 Voice Input
- 🔊 Text-to-Speech Answers
- 🌍 Multi-Language Support
- 🧮 Math Equation Rendering
- 💻 Programming Code Support
- 📑 Markdown Rendering
- 📚 Quiz Generator
- 🧠 Flashcards
- 📄 Revision Notes
- 📊 Formula Sheets
- 💾 Local History Storage
- 🌙 Dark Mode
- 📱 Fully Responsive
- ⚡ Fast Performance
- ✨ Futuristic Animated UI

## 🔑 Gemini API Setup

1. Visit Google AI Studio.
2. Create a free Gemini API Key.
3. Open the application.
4. Paste your API Key.
5. Click **Save**.
6. Start solving homework.
